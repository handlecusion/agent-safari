# Roadmap

The maintained phase roadmap lives in `docs/DEVELOPMENT_PHASES.md`.

Use it together with:

- product vision: `docs/PRODUCT_VISION.md`
- product spec: `docs/PRODUCT_SPEC.md`
- release readiness: `docs/RELEASE_CHECKLIST.md`

Current high-level sequence:

1. Phase 2 — finish native click/actionability hardening.
2. Phase 3 — capture and inspection metadata.
3. Phase 4 — network capture hardening.
4. Phase 5 — session/tab/profile model after single-WebView semantics are stable.

Passkey/WebAuthn automation is explicitly out of scope for the current roadmap.
